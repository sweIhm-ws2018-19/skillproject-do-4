- Name: Alibi erstellen
- Kurzbeschreibung: Erstellung eines Alibis auf Basis der vom User-gelieferten Informationen
- Akteure: User
- Auslöser: "Alexa gib mir ein Alibi" / "Alexa hilf mir" / etc.
- Vorbedingungen: Alexa im Bereitschaftsmodus
- Input: Zeitperiode
	- optional: soziale Umgebung des Users, Aktivitätenraum, Zielperson, räumliche Umgebung des Users
- Ergebnisse: 
	mindestens ein konkretes Alibi für den angegebenen Zeitraum
- Nachbedingungen: 
	Alexa beendet
- Ablauf: 
	- Auslöser durch den User
	- Frage nach dem Zeitraum
	- Speichern des Zeitraums
	- Frage nach weiteren Kriterien
	- Speichern der optionalen Informationen
	- Erstellung des Alibis
	- Rückgabe der Alibi-Ergebnisse
	- Beenden des Skills







- Name: Hilfe
- Kurzbeschreibung: Hilfe-Funktion des Skills 
- Akteure: User
- Auslöser: "Alexa was machst du"
- Vorbedingungen: Alexa im Bereitschaftsmodus
- Ergebnisse: Erklärung des Alibi-Skills
- Nachbedingungen: Alexa im Bereitschaftsmodus
- Ablauf: 
	- Auslöser durch den User
	- Rückgabe einer festen Beschreibung







- Name: Beenden
- Kurzbeschreibung: Beenden des Skills
- Akteure: User
- Auslöser: "Alexa beende Alibi"
- Vorbedingungen: Alexa im Bereitschaftsmodus
- Nachbedingungen: Alexa beendet
- Ablauf: 
	- Auslöser durch den User
	- Beenden von Alexa
